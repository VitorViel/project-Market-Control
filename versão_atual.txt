
📦 PROJETO: MARKET-CONTROL
==========================

📝 DESCRIÇÃO GERAL
------------------
Sistema de controle de mercado com:
- Login e registro de usuários
- Autenticação via JWT
- Cadastro, edição e remoção de produtos
- Painel único para vendedores e admins
- Admins podem promover usuários e deletar outros usuários
- UI responsiva com Tailwind e React 18 (Vite)

📄 README - Alterações Market-Control (Versão 1.5)

📌 Base
- Backend: Versão 1.0 estável (sem alterações)
- Frontend: Atualizado com melhorias visuais e estruturais

✅ Melhorias Aplicadas no Frontend

Interface e Experiência do Usuário
- 🌗 Modo Claro/Escuro: Adicionado botão de alternância de tema (ícones de lua/sol) com persistência automática.
- 🎞️ Animações Suaves: Implementadas animações de entrada/saída usando Framer Motion para páginas e popups.
- 🚪 Transições de Páginas: Todas as rotas (Login, Register, Dashboard) com transição suave.
- 💬 Sistema de Toasts Melhorado: Feedback de ações agora exibido de forma centralizada e estilizada.
- 🔍 AutoFocus em Popups: Campo ID do produto recebe foco automático ao abrir a janela de cadastro/edição.
- 🦴 Skeleton Loaders: Indicadores visuais durante carregamento de produtos e usuários.
- 🖋️ Ícones em Ações: Botões de Editar (✏️) e Deletar (🗑️) com hover animado nos produtos e usuários.
- 🧩 Responsividade Aprimorada: Layout adaptado para dispositivos menores, incluindo posicionamento do botão de tema.

Estrutura de Código
- 🧱 Projeto Frontend migrado para Vite + React 18 + TypeScript + TailwindCSS
- 🔗 React Router DOM implementado: Rotas entre páginas configuradas com animações.
- 🎨 DarkMode controlado por next-themes: Persistência de tema no navegador.
- 📁 Organização Modular: Componentes criados para ThemeToggle, PageWrapper, e melhorias de estilização.
- ⚡ TailwindCSS otimizado: Uso de classes utilitárias modernas no projeto.
